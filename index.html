Depth-Based Outlier Detection

Kernelized Spatial Depth in High-Dimensional Data

A structured exploration of robust statistical depth methods, kernel extensions, and their application to modern data analysis.

Project Overview

This project investigates depth-based approaches to outlier detection, with a focus on Kernelized Spatial Depth (KSD) and its ability to capture nonlinear structure in high-dimensional data.

We bridge theoretical foundations from robust statistics with modern kernel-based methods, culminating in practical implementation and analysis.

Core idea: Transform spatial depth into a feature space using kernels, enabling detection of complex, non-convex outliers.
