Depth-Based Outlier Detection

Kernelized Spatial Depth in High-Dimensional Data

Identifying hidden metabolic risk using depth-based methods on real-world healthcare data.

Project Overview

This project studies multivariate depth-based methods for outlier detection, focusing on Mahalanobis Depth and Kernelized Spatial Depth (KSD).

We apply these methods to NHANES healthcare data, restricting attention to individuals with normal BMI to uncover hidden metabolic risk not detected by conventional approaches.

Key result: KSD consistently identifies individuals with significantly higher HbA1c levels than those detected by Mahalanobis depth (Wilcoxon test, p < 0.05 across all datasets).

Methodological Framework

  • Standardization of clinical variables (glucose, HbA1c, triglycerides, HDL)
  • Outlier detection using Mahalanobis Distance (95th percentile)
  • Outlier detection using Kernelized Spatial Depth (bottom 5%)
  • Comparison of discordant outliers (KSD-only vs MD-only)
  • Statistical validation using one-sided Wilcoxon rank-sum test
Core idea: Kernelization allows depth functions to capture nonlinear structure, enabling detection of clinically meaningful outliers missed by distance-based methods.
