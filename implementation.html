Implementation

From Theory to Data

Applying Kernelized Spatial Depth to real datasets and analyzing outlier behavior in high-dimensional spaces.

Dataset

The dataset for this project is currently under selection. It will be chosen to reflect meaningful real-world structure and allow for clear evaluation of outlier detection methods.

Goal: Identify a dataset where classical distance-based methods fail, but depth-based approaches (especially KSD) provide improved insight.

Methodology

Step 1 — Preprocessing

  • Standardization of features
  • Handling missing values
  • Dimensionality considerations

Step 2 — Spatial Depth

Compute baseline spatial depth:

\( D(x; X) = 1 - \left\| \frac{1}{n} \sum S(y - x) \right\| \)

Step 3 — Kernelization

Apply kernel transformation to capture nonlinear structure.

Step 4 — Outlier Detection

  • Compute depth values
  • Define threshold
  • Classify low-depth points as outliers

Planned Analysis

  • Comparison with classical distance-based methods
  • Behavior in high-dimensional settings
  • Effect of kernel choice
  • Sensitivity to threshold selection
Key focus: Demonstrating how KSD captures local structure that traditional methods fail to detect.
